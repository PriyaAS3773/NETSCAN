# NetScan - Fast TCP Port Scanner

NetScan is a beginner-friendly cybersecurity project developed using Python.  
It scans common TCP ports on a target system and identifies open services.

---

## Features

- Fast TCP Port Scanning
- Multi-threaded Scanning
- Service Detection
- Scan Report Generation
- Error Handling
- Open Port Detection
- Saves Results to Text File

---

## Technologies Used

- Python
- Socket Programming
- Multithreading

---

## How It Works

1. User enters target website or IP address
2. Program converts domain to IP
3. Scanner checks common TCP ports
4. Open ports are displayed
5. Scan report can be saved as a text file
